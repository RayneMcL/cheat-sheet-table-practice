HTML List Reference

List Tags

Tag Name Description
<ul> Unordered List The wrapper element for unordered lists, lists will have bullet points.
<ol> Ordered List The wrapper element for ordered list, lists will be numbered.
<li> List Items Defines a list item, list content.
<dl> Description List Wrapper element defines a description list used in conjuction with <dt> and <dd>.
<dt> Define Term Defines a term/name in a description list.
<dd> Provide Description Decribes a term/name in a description list. You can put paragraphs, line breaks, images, links, lists, etc.
